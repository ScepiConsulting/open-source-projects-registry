## What Was This?

This was the original open-source version of Sticky – a digital sticky note application for organizing thoughts, brainstorming, and real-time collaboration.

![Board](https://github.com/hamzasaleem2/sticky/blob/main/public/board.png)

### Original Features:
- Create and organize digital sticky notes
- Collaborate with others in real-time
- Customize note colors and sizes
- Drag and drop interface
- Cloud sync across devices

## Why Archived?

Sticky started as an open-source project and has since grown into a full product. To focus on building the best experience for users, development has moved to a private repository.

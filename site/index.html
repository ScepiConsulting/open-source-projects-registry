Self-Hosted Knowledge Registry

This site is generated by GitHub Actions.

Start with knowledge/manifest.json or the linked index pages below.

  • knowledge/manifest.json
  • knowledge/ (entry docs)
  • knowledge/apps/ (chunk index)
  • knowledge/repos/ (chunk index)
  • knowledge/readmes/ (README index)
  • knowledge/meta/ (reports)
  • knowledge/meta/fetch_report.json

How To Digest In ChatGPT

  1. Load knowledge/manifest.json.
  2. Load app chunk files from manifest.files.apps.
  3. Use each app's readme_path to fetch full README files.
  4. Use tag/license index files for filtering and recommendations.

Use https://... URLs with GitHub Pages.
